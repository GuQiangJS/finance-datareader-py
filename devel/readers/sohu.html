class finance_datareader_py.sohu.FinancialIndicatorReader(symbols=None, retry_count=3, pause=1, session=None, chunksize=25)¶

从 Sohu 读取主要财务指标

从1990-01-01开始获取数据。

参数:
  • symbols – 股票代码。此参数只接收单一股票代码。For example:600001,000002,300002
  • retry_count – 重试次数
  • pause – 重试间隔时间
  • session –
  • chunksize –
返回:

结果按照 公告日期 倒序 排序。 任意数据表无数据时返回 空白的 pandas.DataFrame 。 参见 pandas.DataFrame.empty。
返回类型:

DataFrame

Examples

>>> df = FinancialIndicatorReader('601398').read()

class finance_datareader_py.sohu.daily.SohuDailyReader(symbols=None, prefix='cn_', start=datetime.date(2004, 10, 8), end=datetime.date(2018, 8, 23), retry_count=3, pause=1, session=None, chunksize=25)¶

从sohu读取每日成交汇总数据

参数:
  • symbols – 股票代码。此参数只接收单一股票代码。For example:600001,000002,300002
  • prefix – 读取股票数据时需要拼接的前缀。默认为 cn_。如果是获取指数时需要使用 zs_。
  • start – 开始日期。默认值：2004-10-08
  • end – 结束日期。默认值：当前日期的 前一天 。
  • retry_count – 重试次数
  • pause – 重试间隔时间
  • session –
  • chunksize –
read()¶

读取数据

返回:pandas.DataFrame 实例。

成交量的单位为 手，成交金额的单位为 万元。

无数据时返回空白的 pandas.DataFrame 。参见 pandas.DataFrame.empty。

>>> from finance_datareader_py.sohu.daily import SohuDailyReader

>>> df = SohuDailyReader(symbols='000002').read()
