Gtimg¶

class finance_datareader_py.gtimg.daily_details.GtimgDailyDetailsReader(symbols: str = None, start=datetime.date(2004, 10, 8), end=datetime.date(2018, 8, 2), retry_count=3, pause=1, session=None, chunksize=25)¶

参数:

  • start – 开始日期。默认值：2004-10-08
  • end – 结束日期。默认值：当前日期的 前一天 。
  • retry_count – 重试次数
  • pause – 重试间隔时间
  • session –
  • chunksize –
read()¶

读取数据

返回:无数据时返回空白的 pandas.DataFrame 。参见 pandas.DataFrame.empty。
返回类型:pandas.DataFrame

Examples

from finance_datareader_py.gtimg.daily_details import GtimgDailyDetailsReader

df = GtimgDailyDetailsReader(symbols='000002', start=datetime.date(2018, 7, 2),
                             end=datetime.date(2018, 7, 2)).read()

class finance_datareader_py.gtimg.daily.GtimgDailyReader(symbols=None, type=None, start=datetime.date(2004, 10, 8), end=datetime.date(2018, 8, 2), retry_count=3, pause=1, session=None, chunksize=25)¶

从 gtimg 读取每日成交汇总数据（支持获取前复权、后复权的数据）

参数:
  • symbols – 股票代码。此参数只接收单一股票代码。For example:600001,000002
  • type –

    {None, ‘qfq’, ‘hfq’}, 默认值 None

    • None: 不复权（默认）
    • ’qfq’: 前复权
    • ’hfq’: 后复权
  • start – 开始日期。默认值：2004-10-08
  • end – 结束日期。默认值：当前日期的 前一天 。
  • retry_count – 重试次数
  • pause – 重试间隔时间
  • session –
  • chunksize –
read()¶

读取数据

返回:无数据时返回空白的 pandas.DataFrame 。参见 pandas.DataFrame.empty。
返回类型:pandas.DataFrame

Examples

from finance_datareader_py.gtimg.daily import GtimgDailyReader

df = GtimgDailyReader(symbols='000002').read()
